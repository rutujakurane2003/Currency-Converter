# Currency Converter

A responsive Currency Converter web application built using HTML, CSS, and JavaScript.

## Features
- Real-time currency conversion
- Dynamic currency selection
- Country flag integration
- Input validation
- Error handling using try-catch
- Responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Fetch API
- Exchange Rate API

## How It Works
1. Enter the amount.
2. Select source currency.
3. Select target currency.
4. Click "Get Exchange Rate".
5. View the converted amount instantly.

## Key Concepts Implemented
- Async/Await
- API Integration
- DOM Manipulation
- Event Handling
- Error Handling
